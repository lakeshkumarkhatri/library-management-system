## Java Swing + DBMS Library Management System

This is a GUI-based Library Management System developed using Java Swing
with MySQL database integration as part of my second semester academic project.

### Features
- Graphical User Interface using Java Swing
- Add, search, update, and delete books
- MySQL database connectivity using JDBC
- Persistent data storage using SQL
- Image-based GUI elements

### Technologies Used
- Java
- Java Swing
- MySQL
- JDBC

### Notes
- Multiple classes are implemented within a single Java file
- MySQL Connector/J is required to run this project

Status: Semester 2 Academic Project
